This folder contains the master file XmlSynchro.Run.exe.XML, the executable XmlSynchro.Run.exe, the library XmlSynchro.dll and the referenced Microsoft CRM SDK files. 

The execution of XmlSynchro.Run.exe, triggered manually or by a scheduled task, reads the master file XmlSynchro.Run.exe.XML and performs the sycnrhonisation processes defined in the <XMLIntegrationFile> nodes. 

.\LOG is configured to held the log files generated by each synchronisation process.
