This folder contains three synchronisation process: 110.USERS.to.CONTACT.xml, 120.RELATIONS.to.RELATIONS.xml and 130.RELUSER.to.RELUSER.xml. 

Bi-directionally synchronises users and contacts between an Oracle database and a Dynamics CRM online organization.

One-way only (Oracle to CRM) integrates the relations reference data table and the relation between the relations referenced data and the users/contacts.
